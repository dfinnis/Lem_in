## Getting Started

First clone this repo to your machine.

```git clone https://github.com/dfinnis/Lem_in.git; cd Lem_in```

Make the binary *lem-in*.

```make```

Run *lem-in* with a map redirected into stdin. You'll find example maps in maps/.

```./lem-in < maps/1.map```
